# ⚖️ LegalSangam

A modern web platform that simplifies **legal consultations** by helping users find advocates based on **location, cost, and ratings**, and providing instant support via a built-in **chatbot**.

🔗 **Live Demo:** [https://legalsangam.web.app](https://legalsangam.web.app)  
🛠️ **Firebase Console:** [LegalSangam Project](https://console.firebase.google.com/project/legalsangam/overview)

---

## 📖 Overview
LegalSangam connects citizens with trusted advocates through a simple, transparent, and accessible platform. Users can search, compare, and book consultations, while also accessing a community hub for legal awareness.

---

## 🛠 Tech Stack
- **Frontend:** React + TypeScript + Tailwind CSS  
- **Backend/Hosting:** Firebase (Auth, Hosting, Firestore optional, JS SDK)  
- **Maps:** React Leaflet  
- **Build Tool:** Vite  

---

## ✨ Features
- 🔐 Firebase **Authentication**  
- 👩‍⚖️ Advocate listings (location, fee, rating, reviews)  
- 🤖 **Chatbot (AskCounsel)** for quick legal queries  
- 📱 **Responsive UI** with Tailwind  
- ☁️ **Firebase Hosting** deployment  

---

## ⚙️ Setup & Run
```bash
# Clone the repo
git clone https://github.com/your-username/legal-sangam.git
cd legal-sangam

# Install dependencies
npm install

# Start locally
npm run dev

# Build for production
npm run build
```

---

## 🚀 Deploy to Firebase
```bash
npm install -g firebase-tools
firebase login
firebase init        # choose Hosting, link project, set dist as build dir
npm run build
firebase deploy
```
